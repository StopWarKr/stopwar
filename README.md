# 🖐StopWar
- 우크라이나 전쟁관련 소식 제공 비영리 서비스
- 서비스 링크 : www.stopwar.co.kr

## 목적
- 플랫폼 별 **다양한 전쟁 관련 뉴스** 를 항목별 아카이빙하여 전쟁 상황을 한 눈에 파악
- 우크라이나에 **금전적으로 후원할 수 있는 수단**을 아카이빙하여 후원에 대한 접근성 향상
